m move attractors and cannons
+ increase the rate at which cannons fire and the mass of attractors
- decrease the rate at which cannons fire and the mass of attractors
k delete the currently select object
t turn a cannon to point at the current mouse position
shift+b delete all bodies on screen
shift+c delete all cannons on screen
shift+a delete all attractors on screen
shift+d delete all deflectors on screen
a create an attractor at the current mouse coordinate
b create a body at the current mouse coordinate
c create a cannon at the current mouse coordinate
d create a deflector at the current mouse coordinate
left arrow key moves selected object left
right arrow key moves selected object right
up arrow key moves selected object up
down arrow key moves selected object down
